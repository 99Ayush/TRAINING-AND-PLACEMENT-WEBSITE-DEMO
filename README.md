# 🎓 CampusSync


**CampusSync** is a premium, next-generation Student Placement Portal designed to streamline the recruitment lifecycle for both students and Training & Placement Officers (TPO).

---

## ✨ Features

- **🚀 Live Recruitment Drives**: Real-time "Ongoing Drives" board with eligibility checks and one-click applications.
- **📊 Student Dashboard**: Personalized analytics, CGPA tracking, and automated recruitment pipeline status.
- **📂 Application Pipeline**: A dedicated space to track every step of your professional journey from initial screening to offer.
- **👔 Partner Nexus**: Administrative interface for managing corporate partners and scheduling interviews.
- **💎 Premium UX**: Sleek glassmorphic interface built with Framer Motion for a state-of-the-art feel.

## 🛠️ Tech Stack

- **Frontend**: React + Vite
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Styling**: Vanilla CSS (Custom Glassmorphism System)
- **State Management**: React Hooks & Local Storage Persistence

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/TRAINING-AND-PLACEMENT-WEBSITE-DEMO.git

# Navigate to frontend
cd DBMS_PROJECT/frontend

# Install dependencies
npm install

# Start development server
npm run dev
```
